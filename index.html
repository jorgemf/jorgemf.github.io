Java utilities (under development v0.1)

Set of utilities in java.

  • Data structures:
    • trie
    • b-star (memory based)
    • octree + basic intersection shapes
  • Search algorithms:
    • a-star (normal and parallel version)
    • genetic algorithm (normal and parallel version)

Android loading helper (v1.0)

Helper classes to perform pull to refresh and endless scrolling in a recycler view.

Add it to your android project in the build.gradle file:

dependencies {
    compile 'com.livae:android-loadingHelper-v1.0'
}
